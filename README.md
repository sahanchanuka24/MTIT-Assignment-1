📝 Project Overview
This project explores the practical application of Generative AI (GenAI) tools within the modern Information Technology landscape. As a Junior AI Technologist, the focus of this assignment is to demonstrate how GenAI can enhance core software engineering tasks, specifically focusing on productivity and efficiency.

For this assignment, Option A: Software Engineering was selected.

🎯 Objectives
The primary objective is to experiment with and evaluate two prominent GenAI tools by applying them to a functional IT problem:

ChatGPT

Claude AI

🛠️ Project Scope
The implementation covers the following key areas:

Code Generation: Developed a functional Login System module using Python Flask and SQLite database.

Debugging: Identifying and resolving errors within existing code snippets to ensure system stability.

Documentation: Creating comprehensive technical documentation for the developed module.

Comparative Evaluation: Analyzing both tools based on output quality, prompt sensitivity, and technical limitations.

💻 Tech Stack
Backend: Python Flask

Database: SQLite (database.db)

Frontend: HTML (Jinja2 Templates)

Security: Flask-Bcrypt (Hashing), Flask-WTF (CSRF Protection)

Tools: VS Code, Postman, Git/GitHub

🚀 Key Features
User Registration with password hashing.

Secure User Login and Session Management.

Protected Dashboard access.

CSRF protection for form submissions.

Custom 404 and 500 Error Handling.
